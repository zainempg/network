NetworkModuleBuilder

class NetworkModuleBuilder

A builder class for creating and configuring a Retrofit instance.

This class provides a fluent API for adding various components to the Retrofit instance, such as interceptors, converters, call adapters, and network configurations.

Usage:

val retrofit = NetworkModuleBuilder()
    .addInterceptor(HttpLoggingInterceptor().apply { level = HttpLoggingInterceptor.Level.BODY })
    .addConverterFactory(GsonConverterFactory.create())
    .buildRetrofit("https://api.example.com/")
